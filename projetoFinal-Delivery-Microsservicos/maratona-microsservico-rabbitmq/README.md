## Descrição

RabbitMQ usado entre os microserviços da Maratona FullCycle.

## Rodar a aplicação

#### Crie os containers com Docker

```bash
$ docker-compose up
```

#### Accesse a UI do RabbitMQ no browser

```
http://localhost:15672

User: admin
Password: admin
```
